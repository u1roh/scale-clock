# 音楽のスケール変換早見ダイヤルアプリの開発

## 1. 目的
- 目的: 例えばピアノ用の楽譜のスケールをアルトサックス用のスケールにパッと変換できるような早見表をつくること。
- 背景:
  - 私はアルトサックスを習っているが、音楽理論に習熟していないのでスケール変換に苦労する。
  - アルトサックスはE♭管なので、サックスでCを吹くと、実際にはE♭の音が鳴る。従って、他の楽器と合わせる時には楽譜のスケール変換が必要になる。
  - 「この楽譜はピアノ用の楽譜でGメジャースケールだから、これをアルトサックスで合わせるには、ええっと、ええっと・・・？」と軽くパニックになるので、簡単な早見表が欲しい。

## 2. 対象範囲
- すべて自由に編集して良い

## 3. 要件
- 機能要件:
  - ブラウザで動くこと
  - １オクターブに半音階は12音あるので、これを時計のように円形に並べて表示すること。
    - 12音階 = C, C#/D♭, D, D#/E♭, E, F, F#/G♭, ...
    - 時計の文字盤には、ピアノの白鍵に相当する音は C, D, E といった文字を表示すること
    - 一方、ピアノの黒鍵に相当する音は黒く塗りつぶすこと
  - 1オクターブの円盤は外周と内周の2つがあり、同心円状に重ねて表示すること
  - 内側の円盤は何らかの操作によって回転できること。
- 非機能要件:
  - とくになし

## 4. 制約
- 使用技術の制約:
  - SVGで表現できると良い
